	Projeto em Python para automatizar a evolução dos Pokémon dentro do jogo. 

	O script permite que o personagem se mova automaticamente em busca de oponentes, interagindo com o ambiente do jogo. Ele monitora constantemente se há combates disponíveis e, ao encontrar um, executa sequências de comandos de ataque até que o oponente seja derrotado. Após o combate encerrar ele volta ao processo de procura. Isso resulta em ganho de experiência para o personagem, possibilitando seu avanço de nível conforme participa dos combates.
	Obs: Script em produção/teste.
